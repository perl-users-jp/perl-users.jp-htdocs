Perl5でできたプロダクトを紹介します。

GrowthForecast

GrowthForecastは様々な値をWebAPI経由でグラフ化できるWebツールです。シンプルなAPIでリアルタイムにグラフを作成、更新でき、Webインターフェイスからグラフの表示をカスタマイズしたり、複数のメトリクスを重ね合わせたグラフを作成できます

GNU Parallel

GNU Parallel は1つまたは複数のコンピューターでジョブを並行動作させるためのシェルツールです。

GNU Parallelがすごすぎて生きるのがつらい

STF

STFはMySQL, Perl, memcached等を利用した分散オブジェクトストアです。何億個、何十億個という単位でファイルを格納可能。各ファイルは自動的に複製されデータロスを防ぎます。livedoorBlog等で画像ファイルを格納・配信するために利用されています。解説記事・STFのワーカー自律分散

MHA for mysql

MySQL マスターのHA化を行うためのツールです。

最小限のダウンタイムでデータの不整合を防ぎつつ、マスターのフェイルオーバーを行います。

Ikachan

botによるIRCへの自動投稿をHTTPサーバーを通すことによりIRC機能を実装する必要無く実現させてくれるプロキシ。ジョブの終了などの通知をHTTP POSTを送るだけでIRCに周知することが可能になり捗ります

Cinnamon

role/taskという枠組みと、コマンドをローカル/リモートで実行するという仕組みだけを提供する、シンプルなデプロイツールです。どんなプロダクトにも柔軟に対応できる、使いやすいツールです。
